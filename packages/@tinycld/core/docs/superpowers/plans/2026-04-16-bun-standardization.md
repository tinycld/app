# Bun Standardization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Eliminate the hybrid npm/bun setup by standardizing fully on Bun — consistent lockfile, scripts, and dependency resolution.

**Architecture:** Replace the binary `bun.lockb` with text-based `bun.lock`, swap all `npm run`/`npx` references to `bun run`/`bunx`, remove the `.npmrc` workaround, and pin the Bun version via `packageManager` field.

**Tech Stack:** Bun 1.1.43, Expo EAS Build

### Task 2: Replace npm/npx with bun/bunx in package.json scripts

**Files:**
- Modify: `package.json` (lines 12–38, scripts section)

- [ ] **Step 1: Replace all `npx` with `bunx` and `npm run` with `bun run` in scripts**

Apply these exact replacements in the `"scripts"` section of `package.json`:

```json
"scripts": {
    "packages:generate": "bunx tsx scripts/generate-packages.ts",
    "predev": "bun run packages:generate",
    "prebuild:web": "bun run packages:generate",
    "android": "bunx expo run:android",
    "migrate": "./server/tinycld migrate",
    "build:web": "bunx expo export --platform web",
    "dev": "run-p -r ssl:dev expo:dev pb:dev",
    "pb:dev": "set -a && [ -f .env ] && . .env && set +a && cd server && go build && cd - && exec ./server/tinycld --dev --http 0.0.0.0:7093 serve",
    "ssl:dev": "local-ssl-proxy  --source 7090 --target 7093 --cert ./assets/localhost.pem --key ./assets/localhost-key.pem",
    "dev:local": "run-p -l -r expo:dev ssl:dev pb:dev",
    "expo:dev": "EXPO_PUBLIC_ENV=dev bunx expo start --port 7100",
    "expo:test": "EXPO_PUBLIC_ENV=test bunx expo start --port 7101",
    "dev:clean": "bunx expo start --port 7100 --clear",
    "ios:simulator": "EXPO_PUBLIC_ENV=dev bunx expo run:ios --device B5B76CBA-B42A-4C06-9D55-9B391BB4175F",
    "ios": "bunx expo run:ios",
    "prebuild:native": "bunx expo prebuild",
    "lint": "biome check .",
    "lint:fix": "biome check --write .",
    "typecheck": "tsc --noEmit --skipLibCheck",
    "checks": "bun run lint && bun run typecheck",
    "eas:submit:ios": "bunx eas submit --platform ios",
    "db:reset": "bunx tsx scripts/reset-dev-db.ts",
    "db:seed": "bunx tsx scripts/seed-db.ts",
    "test:e2e:debug": "playwright test --ui --headed --debug",
    "test:e2e": "bunx playwright test",
    "test:unit": "bunx vitest run",
    "test:server": "bunx tsx scripts/test-server-api.ts",
    "push:genkeys": "cd server/push && go run ./cmd/genkeys"
},
```

Changes summary:
- `npx tsx` → `bunx tsx` (3 scripts: packages:generate, db:reset, db:seed, test:server)
- `npx expo` → `bunx expo` (7 scripts: android, build:web, expo:dev, expo:test, dev:clean, ios:simulator, ios, prebuild:native)
- `npm run` → `bun run` (3 scripts: predev, prebuild:web, checks)
- `npx eas` → `bunx eas` (1 script: eas:submit:ios)
- `npx playwright` → `bunx playwright` (1 script: test:e2e)
- `npx vitest` → `bunx vitest` (1 script: test:unit)

- [ ] **Step 2: Verify scripts parse correctly**

```bash

```

Expected: lint and typecheck both pass (same as before — this is a script runner change, not a code change).
