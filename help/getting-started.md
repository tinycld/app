## Welcome

This app is built from a small **core** plus a set of optional **feature packages** — mail, contacts, calendar, drive, and more. Each package adds its own screens, settings, and data. You only see what you have installed.

## The main areas

- **Nav rail** — the column on the left lists every installed feature package and a handful of always-present items (settings, help). Click a package icon to jump to its main screen.
- **Sidebar** — most packages have a contextual sidebar to the right of the rail. It holds folders, filters, or other navigation specific to that package.
- **Workspace** — the main area where the package renders its current screen.

## Keyboard shortcuts

Press `t` followed by a single letter to jump between packages (the letter is shown in each package's docs). Press `?` anywhere to open this help.
